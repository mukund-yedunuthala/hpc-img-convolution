template<typename kernel>
void convolution(int *&valueArray, int *&resultArray, kernel **&operation, unsigned int &rows, unsigned int &cols, unsigned short int &size)

Performs convolution parameter with a given kernel on a given array.

Parameters:

  • valueArray – Input array.

  • resultArray – Output array.

  • operation – Convolution kernel.

  • rows – Number of rows present in input array.

  • cols – Number of columns present in input array.

  • size – Size of the convolution kernel array.
